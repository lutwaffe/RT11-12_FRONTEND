# Практические занятия 9–12: Frontend & Backend разработка

## Описание

Полноценное веб-приложение с аутентификацией по JWT, системой ролей (RBAC) и управлением товарами. Бэкенд реализован на Node.js + Express, фронтенд — на React.js.

---

## Структура репозитория

```
RT_9/          — Практика 9: JWT + Refresh-токены (только сервер)
RT_10/
  server/      — Практика 10: REST API с продуктами
  client/      — Практика 10: React-фронтенд с localStorage и axios interceptors
RT_11/
  server/      — Практика 11: RBAC (роли: user, seller, admin)
  client/      — Практика 11: React-фронтенд с ролевым доступом
```

---

## Запуск

### Сервер

```bash
cd RT_11/server
npm install
npm start
```

Сервер запустится на `http://localhost:3000`

### Клиент

```bash
cd RT_11/client
npm install
npm start
```

Клиент запустится на `http://localhost:3001`

## API маршруты

### Аутентификация

| Метод | Маршрут              | Доступ        | Описание                       |
|-------|----------------------|---------------|--------------------------------|
| POST  | /api/auth/register   | Гость         | Регистрация пользователя       |
| POST  | /api/auth/login      | Гость         | Вход в систему                 |
| POST  | /api/auth/refresh    | Гость         | Обновление пары токенов        |
| GET   | /api/auth/me         | Пользователь  | Получение текущего пользователя|

### Пользователи

| Метод  | Маршрут          | Доступ         | Описание                        |
|--------|------------------|----------------|---------------------------------|
| GET    | /api/users       | Администратор  | Список пользователей            |
| GET    | /api/users/:id   | Администратор  | Пользователь по id              |
| PUT    | /api/users/:id   | Администратор  | Обновить информацию             |
| DELETE | /api/users/:id   | Администратор  | Заблокировать пользователя      |

### Товары

| Метод  | Маршрут              | Доступ              | Описание               |
|--------|----------------------|---------------------|------------------------|
| GET    | /api/products        | Пользователь+       | Список товаров         |
| POST   | /api/products        | Продавец, Админ     | Создать товар          |
| GET    | /api/products/:id    | Пользователь+       | Товар по id            |
| PUT    | /api/products/:id    | Продавец, Админ     | Обновить товар         |
| DELETE | /api/products/:id    | Администратор       | Удалить товар          |

---

## Роли

| Роль    | Описание                                              |
|---------|-------------------------------------------------------|
| `user`    | Только просмотр товаров                             |
| `seller`  | Просмотр + создание и редактирование товаров        |
| `admin`   | Все права + удаление товаров + управление юзерами   |

---

## Технологии

- **Backend:** Node.js, Express, bcrypt, jsonwebtoken
- **Frontend:** React.js, React Router v6, Axios
- **Аутентификация:** JWT (access + refresh токены с ротацией)
- **Авторизация:** RBAC middleware
